# iTerm2ToXShell

iTerm2ToXShell is a toolkit for converting the iTerm2 Color Schemes to XShell format

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'iTerm2ToXShell'
```

## Usage

Usage:

	iTerm2ToXShell [Options]

Options:

	-b, [--base-dir]	# Base directory

	-v, [--version]		# Version
